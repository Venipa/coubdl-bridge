- Change to Release Build
- CTRL + B to build

## Install / Register

Windows (x86)
```asciidoc
- Download Release rev...zip
- Unpack it to your desired folder
- open destination folder
- run cmd in folder
- type "CoubDownloadBridge.exe -register" in the running cmd
- approve the UAC Request
```
